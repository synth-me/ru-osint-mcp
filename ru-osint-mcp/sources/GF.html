General overview of barracks used by Russian Ground Forces.


The aim of this general overview is to analyze and document the infrastructural features of these facilities using open sources such as Google Earth and make them available as open source intelligence (OSInt).

The project >Ground Forces< is currently being developed. Before the complete filling of the database is possible, the structure of the database and the output formats are first developed and tested. The data will then be filled in step by step. The project >Ground Forces< is therefore a prototype that is still subject to change.
